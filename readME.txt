This repository contains hands‑on lab exercises for an introductory security practical course (brute‑force, keylogging in a lab, log analysis, etc.).

Requirements

Recommended OS / environment
-Ubuntu 22.04 LTS or later, or WSL (Ubuntu) on Windows. (Labs were written for Debian/Ubuntu derivatives.)
-Enough disk space for wordlists (if you download large lists).

Software

-Python 3.10+ (3.8+ may work for some scripts)
-pip
-virtualenv (or python -m venv)
